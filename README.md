## 📖 Як користуватися репозиторієм

Цей репозиторій розділений на незалежні модулі. Кожен проєкт знаходиться у власній папці та містить:

1. **Індивідуальний README**: технічні вимоги (версії ПЗ, залізо), інструкція із запуску та методичні підказки.
2. **Вихідний код**: повністю робочий проєкт із коментарями.
3. **Assets**: необхідні зображення, звуки та текстури.

Для початку роботи оберіть майстер-клас, який вас цікавить, у таблиці вище та перейдіть у відповідну папку.

## 👨‍🏫 Про проєкт

Матеріали створені викладачем школи **Robocode** для навчання дітей віком від 5 до 16 років.

**Основна мета:** Вихід за межі простого написання коду. Через гейміфікацію та сучасні технології (AI, CV, Robotics) ми допомагаємо дітям розвивати **стратегічне та алгоритмічне мислення**, вчимо аналізувати складні завдання та знаходити креативні інженерні рішення. Проєкти спрямовані на те, щоб дитина опанувала інструменти майбутнього, навчилася працювати з параметрами реальних пристроїв та розуміла пропорції та логіку побудови цифрових світів.
